# Script Overlay

The **Script Overlay** is a slide-up panel available from any tab. It displays the Python script that corresponds to the current timeline, generated client-side by `timelineToScript.js`.

## Opening the overlay

Click the **`</>`** button in the top bar. The overlay slides up from the bottom of the screen. Click the button again, or press the **×** close button inside the overlay, to dismiss it.

The overlay also opens automatically when a toolpath preview fails — it shows the Python code that would have been run so you can inspect it.

## What it shows

The script is read-only. It is a valid UTDE Python script generated by walking the timeline entries in order, maintaining the active orient chain, and emitting one API call per entry:

```python
from toolpath_engine import *

machine = Machine.load("gantry_5axis_ac")

# Op 1 — Raster fill
faces = [2, 5, 7]
strategy = RasterFillStrategy(spacing=2.0, angle=0, feed_rate=600)
paths = strategy.generate(faces=faces)
paths.orient(to_normal(), lead(10))

# Op 2 — Follow curve
edges = [0]
strategy = FollowCurveStrategy(feed_rate=400)
paths2 = strategy.generate(edges=edges)
paths2.orient(fixed(0, 0, -1))
```

The timeline is the source of truth — editing the script here has no effect. To change the output, modify the timeline entries.

## Toolbar actions

| Button | Action |
|---|---|
| **Copy** | Copies the full script to the clipboard |
| **×** | Closes the overlay |
